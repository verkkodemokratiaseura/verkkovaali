Nuorisovaalit 2011
==================

This is the end user application which implements the voting interface for
Nuorisovaalit 2011. Authenticated users are able to vote for candidates in
their designated voting districts. The application implements the voting
process according to the process diagram below.

.. image:: images/Voting-process-flow.png
